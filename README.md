功能：快速创建一个标准的 typescript 项目代码库

## Usage

全局安装

```
npm install -g @popak/tp-starter
```

命令行创建项目

```
tps create xxx
```

## Features

1. 使用 webpack/rollup 打包
2. 使用 typescript 生成 d.ts 文件
3. 使用 eslint, prettier 规范代码
4. 使用 husky, lint-staged, pre-commit 进行代码提交检查修正
5. 使用 typedoc 生成文档
6. 自动初始化 git 代码库
7. 自动安装依赖库
